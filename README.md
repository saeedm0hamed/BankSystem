A simple CRUD web application built with ASP.NET MVC Core to manage essential banking operations efficiently, developed as part of my Database and Information Systems courses at FCI - ARU.

💡 Key Features:
- Data Management: Full create, read, update, and delete operations for branches, employees, customers, accounts, and transactions.
- Database Integration: Efficient relational database design using SQL Server, and optimized table relationships with Entity Framework Core and Fluent API.
- Transaction Tracking: Keep a record of deposits, withdrawals, and transfers.
- Transaction History: Detailed logs for every account, including date, type, and amount.
- Search Functionality: Quickly find items with keywords.
- Responsive Design: Optimized for both desktop and mobile devices.

⚙️ Technologies & Techniques:
- Backend: ASP.NET Core MVC, 3-Tier Architecture, Dependency Injection
- Frontend: Bootstrap 5, jQuery (+ libraries like FontAwesome, SweetAlert, and Toastr for interactivity and improved UI/UX)
